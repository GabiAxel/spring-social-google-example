Code example:

// List the user's events
EventPage page = google.calendarOperations()
   .eventListQuery(calendarId)
   .fromPage(pageToken)
   .timeMin(new Date(year, month, 1))
   .timeMax(new Date(year, month + 1, 1))
   .orderBy(OrderBy.START_TIME)
   .singleEvents(true)
   .getPage();
